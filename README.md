# Final_Project

This is a text-based adventure game where you and your son go on a journey. You will make choices to explore, fight, or talk to your son. Each choice leads to different results and progresses the story.

Choice-Based Story: Your decisions change the outcome of the game.
Combat: Fight wild animals and monsters with random success.
Exploration: Discover items and secrets as you travel.
Character Interaction: Talk to your son to learn more about his feelings.

1. Download the game files and keep them in the same folder.
2. Open a terminal or command prompt, navigate to the folder, and run
3. Only run the game using main.py. Running other files like chapter1.py directly will not work.
4. Follow the on-screen instructions to start playing.

Make Choices: Choose to explore, fight, or talk at each step.
Random Outcomes: Some actions, like fighting, are based on luck (random chance).
Progress Through Chapters: Winning battles or making good choices takes you to the next chapter.
